Set up your API key
You need a free Anthropic API key to power this app. It only takes 2 minutes.
1
Create a free Anthropic account
Go to console.anthropic.com and sign up. It's free — you only pay for what you use (fractions of a cent per generation).
2
Get your API key
Once logged in, go to API Keys in the sidebar. Click Create Key, give it a name like "content agent", and copy it.
3
Paste your key below
Your key is saved only in your browser — it never leaves your device.
✓ Key saved. You're ready to generate.
Deploy to GitHub Pages (make it a real app)
1
Create a GitHub account
Go to github.com and sign up for free.
2
Create a new repository
Click the + icon → New repository. Name it content-agent. Set it to Public. Click Create.
3
Upload this file
Click uploading an existing file, drag in the index.html file you downloaded, then click Commit changes.
4
Turn on GitHub Pages
Go to your repo → Settings → Pages → set Source to main branch → Save. In ~60 seconds your app is live at yourusername.github.io/content-agent
